6. ИЗВЕСТНЫЕ ПРОБЛЕМЫ
--------------------------------------------------------------------------------

AllTick WebSocket:
  - Токен требует активации на alltick.co
  - Сейчас возвращает 429 Too Many Requests
  - Решение: зарегистрироваться и активировать токен

AllTick REST API:
  - Forex (EURUSD и др.) возвращает 429
  - GOLD работает стабильно
  - Решение: используется Backup API для Forex

Alpha Vantage:
  - Лимит 5 запросов/минуту на бесплатном тарифе
  - Нет real-time данных (задержка до конца trading day)
  - Нет интрадей данных на бесплатном тарифе
  - Решение: кэширование на 60 секунд

SILVER и WTI:
  - Превышение лимита Alpha Vantage при частых запросах
  - Решение: увеличить интервал обновления или добавить Twelve Data

АЛЬТЕРНАТИВЫ ДЛЯ БЕСПЛАТНЫХ REAL-TIME КОТИРОВОК:

1. Twelve Data (бесплатный тариф)
   - 8 запросов в минуту
   - 800 запросов в день
   - Real-time Forex, металлы, акции, крипто
   - Endpoint: https://api.twelvedata.com/price
   - Требуется регистрация: https://twelvedata.com/pricing

2. Binance API (крипта + токенизированные активы)
   - Бесплатно, без лимитов (разумное использование)
   - Real-time данные через WebSocket
   - Есть GOLD, SILVER, нефть (токенизированные)
   - Endpoint: https://api.binance.com/api/v3/ticker/price
   - Не требуется регистрация для public API

3. Financial Modeling Prep (бесплатный тариф)
   - 250 запросов в день
   - Real-time данные для US рынка
   - Forex, акции, крипто
   - Endpoint: https://financialmodelingprep.com/api/v3/quote/
   - Требуется регистрация: https://site.financialmodelingprep.com/developer-pricing

ПРИОРИТЕТ: Twelve Data → Binance → FMP
